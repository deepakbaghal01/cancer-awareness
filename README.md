# Hope Together — Cancer Support & Awareness

A simple, responsive demonstration website providing compassionate support, educational content, and inspirational quotes for people affected by cancer.

## Features
- Clean, responsive UI built with HTML, CSS, and vanilla JavaScript
- Sections: Hero, Awareness, Support, Resources, Contact (demo), Quotes (real-time)
- Dark / Light theme toggle with system preference support
- Real-time inspirational quotes via the Quotable API with local fallback
- Accessible and mobile-friendly navigation and form handling

## Files
- `inde.html` — Main HTML page
- `style.css` — Styles and dark-mode variables
- `javascript.js` — Interactive behavior (menu, form, quotes, theme)
- `animation.js` — Additional animations ()
